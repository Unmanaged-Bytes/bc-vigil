Présentation

BitCrafts Vigil est une interface web autour de deux outils en ligne de commande :

  • bc-hash pour le module Intégrité : hache une arborescence et compare à une baseline pour détecter des dérives (ajouts, suppressions, modifications).
  • bc-duplicate pour le module Doublons : trouve des fichiers identiques, permet de les mettre en quarantaine puis de les restaurer ou de les supprimer définitivement.

Les deux modules partagent la même base de données SQLite, les mêmes planificateurs cron, le même système de sauvegarde/restauration et la même UI FR/EN. Deux badges dans la barre de navigation comptent les alertes non acquittées (dérives d'intégrité et analyses avec doublons).

Prise en main

  1. Créer une cible intégrité et/ou une cible de recherche de doublons (section Cibles pour l'intégrité, Doublons pour la déduplication).
  2. Lancer un premier scan manuel : côté intégrité il devient la baseline; côté doublons il produit un rapport avec les groupes de fichiers identiques.
  3. Ajouter éventuellement un ou plusieurs schedules pour automatiser les scans.
  4. Agir sur les résultats : promouvoir ou acquitter une dérive d'intégrité, mettre en quarantaine des doublons.
