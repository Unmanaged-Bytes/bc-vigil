Overview

BitCrafts Vigil is a web frontend around two command-line tools:

  • bc-hash for the Integrity module: hashes a directory tree and compares with a baseline to detect drifts (added, removed, modified files).
  • bc-duplicate for the Duplicates module: finds identical files, lets you quarantine them and later restore or permanently purge.

Both modules share the same SQLite database, cron schedulers, backup / restore machinery and FR/EN UI. Two badges in the navbar count unacknowledged alerts (integrity drifts and dedup scans with duplicates).

Getting started

  1. Create an integrity target and/or a dedup target (Targets for integrity, Duplicates for dedup).
  2. Run a first manual scan: the integrity side adopts it as the baseline; the dedup side produces a report with duplicate groups.
  3. Optionally add one or more schedules to automate scans.
  4. Act on results: promote or acknowledge an integrity drift, quarantine selected duplicates.
